# Peran Tim

---
## Desainer (DKV)
- Membuat UI/UX di Figma  
- Menyediakan aset (logo, ikon, watermark)  

---
## Developer (RPL)
- Menerjemahkan desain ke kode

---
## PM
- Mengarahkan

